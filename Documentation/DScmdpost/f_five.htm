F5: Refresh


Refresh (also available via F2: 6) completely redraws the hex editor display. The editor will redisplay each byte value on the screen. This feature is useful when you are viewing a volatile memory area, and need the current values, or are unsure if the memory that you are viewing has changed. Note: If the disassembler view is active, the disassembly results will be refreshed instead
