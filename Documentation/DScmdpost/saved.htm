Command Post

Being Developed For:  TI-89 / TI-92 Plus
Expected Release:  December

Produced By: Greg Dietsche
E-mail: greg@detachedsolutions.com
   

This Flash Application for the TI-89 and TI-92 Plus is designed with developers in mind. It has many features including:
  • Hex Editor with many features such as paste special, and search functions.
  • Crash Protection which allows developers to recover important information for debugging
  • Multi-function Low Memory Simulation
  • Leak Watch monitors Flash Applications and reports any change in memory
  • Detailed Information about the calculator's hardware
  • Detailed Information about the software installed on the calc. (Kernel, HWPatch, etc)
  • Six Peek/Poke TI-Basic extensions
  • Shared Code Library
  • Fully Language Localizable
